Cascading Style Sheets

Cascading Style Sheets or CSS is used for coding websites. To be more specific, it allows the coder to change things in the website such as the font of the text, the color of the text, the background color, etc. This is useful in making the website look more streamlined and have a better aesthetic.

Bootstrap

Bootstrap Version 4

Bootstrap is an application that allows you to use Responsive Web Design. Responisve Web Design is used for making websites easier to view and use by allowing them to automatically adjust to fit the size of the screen being used to view it. This is especially useful on mobile devices since they typically have small screens.
